# 财政、地区与粮运系统

靖康危机不是只有战场。汴京能不能守住，取决于钱粮能否入京、军饷能否发下、粮价能否压住、地方是否愿意响应朝廷。

本系统的目标是让玩家理解：

```text
没有钱，圣旨只是纸。
没有粮，汴京就是孤城。
没有道路和执行人，勤王只是奏报上的忠心。
```

## 核心账户

| 账户 | 单位 | 说明 |
| --- | --- | --- |
| 国库 | 万贯 | 朝廷公开财政，用于军饷、城防、赏赐、外交赔款 |
| 内帑 | 万贯 | 宫中私库，用于救急、密支、安抚、秘密收买 |
| 军资库 | 万贯/物资 | 专用于军器、火油、箭矢、甲胄、马匹 |
| 京城粮 | 万石 | 汴京可用粮储，围城阶段核心资源 |

首发可简化为顶栏显示：

```text
国库 / 内帑 / 京城粮
```

军资库在城防详情中显示。

## 钱粮流水

所有财政变化都要走流水。

字段建议：

| 字段 | 说明 |
| --- | --- |
| turn | 回合 |
| account | 国库/内帑/军资库 |
| delta | 变化 |
| balance_after | 变化后余额 |
| category | 税收/军饷/城防/抄没/借款/赔款/赏赐 |
| reason | 原因 |
| source | 来源地区、人物、旨意或事件 |
| visibility | 公开/秘密 |

好处：

- 玩家能追账。
- 殿前对质能引用账本。
- LLM 不能凭空说钱没了。
- 财政失败能产生线索。

## 收入来源

北宋财政不要做成完整经济模拟，首发只保留与靖康危机有关的来源。

| 来源 | 玩法 |
| --- | --- |
| 常赋 | 地区基础收入，受控制度、民心、道路影响 |
| 盐茶酒税 | 可快速增收，但伤民心和商户 |
| 商户借款 | 救急强，但需要信用和抵押 |
| 内帑拨补 | 快速有效，但宗室内廷不满 |
| 抄没追赃 | 爽点来源，但依赖证据，伤派系 |
| 临时加派 | 高收益高民怨，容易引发逃亡和粮价 |
| 外交赔款 | 负向支出，可能换取时间 |

## 固定支出

| 支出 | 说明 |
| --- | --- |
| 禁军军饷 | 欠饷会直接影响守城 |
| 勤王赏格 | 没有赏格，各路军可能观望 |
| 城防修缮 | 城墙、城门、器械、壕沟 |
| 粮食收购 | 平抑京城粮价 |
| 官俸 | 长期拖欠会降低官僚执行 |
| 情报密支 | 皇城司、反间、收买金营 |
| 外交礼物 | 谈判和拖延使用 |

## 地区模型

首发不要铺满全国，重点围绕汴京、北方战线和财税后方。

### 核心地区

| 地区 | 玩法定位 |
| --- | --- |
| 汴京 | 京城、围城主盘面 |
| 京畿 | 汴京外围、粮道、金军进逼 |
| 河北东路 | 金军南下通道、边防压力 |
| 河北西路 | 真定、中山等防线想象空间 |
| 河东路 | 太原屏障，牵制金军关键 |
| 陕西路 | 西军来源，勤王核心 |
| 淮南 | 后撤、粮运、南方防线 |
| 江南/两浙 | 财赋后方、借款、粮食 |
| 荆湖/川峡 | 长线资源与二线安全 |

## 粮运系统

粮运是《朕不南渡》和当前明末项目最大的不同之一。

### 路线

路线不是抽象收入，而是可被攻击、堵塞、贪污、拖延的链条。

首批路线：

- 江淮粮道 → 汴京。
- 京畿短途征粮 → 汴京。
- 陕西军粮 → 河东/汴京。
- 河北撤粮 → 汴京。
- 汴河漕运节点。

每条路线字段：

| 字段 | 说明 |
| --- | --- |
| capacity | 最大运输量 |
| risk | 被截断或损耗风险 |
| delay | 需要几回合到达 |
| controller | 谁在实际控制 |
| corruption | 损耗与截留 |
| escort | 护送力量 |

### 粮运玩法

玩家可以：

- 派转运使催粮。
- 拨禁军或民壮护粮。
- 给商户保价收粮。
- 下令开仓平粜。
- 查办囤积。
- 牺牲某地民心强征。
- 改道绕开金军。

结果可能：

- 粮到汴京。
- 半路被截。
- 地方虚报。
- 粮价被压住。
- 民怨上升。
- 商户囤积转地下。

## 京城粮价

围城题材里，粮价是民心和战意的转换器。

建议指标：

```text
粮价指数：100 = 平稳，200 = 翻倍，300 = 恐慌
```

影响：

- 粮价高 → 民心下降。
- 粮价高 → 城中战意下降。
- 粮价高 → 太学生请愿和商户冲突上升。
- 粮价高 → 主和派更容易说“百姓不可久困”。

玩家应对：

- 开仓。
- 平粜。
- 抓囤积。
- 从江淮调粮。
- 减少军粮消耗。
- 组织民壮换粮票。

## 抄没与追赃

这是财政系统的爽点来源。

### 触发

- 密令查账。
- 事件暴露。
- 殿前对质定案。
- 战败问责。
- 旧党清算。

### 过程

```text
发现亏空 → 查账 → 锁定人物/网络 → 对质 → 裁断 → 抄没/追缴 → 入库
```

### 风险

- 证据不足会被说成罗织。
- 重判会吓坏官僚，短期执行上升，长期怠工。
- 抄商户会打击粮运。
- 抄旧党会触发毁证和外逃。

## 地方响应

地方不是按按钮响应。每个地区对朝廷的响应取决于：

```text
响应 = 君威 + 地方民心 + 承办人能力 + 路线安全 - 敌军压力 - 派系阻力 - 最近加派伤害
```

响应影响：

- 税粮是否到京。
- 勤王军是否出发。
- 地方是否接纳撤退军民。
- 金军绕道时是否抵抗。

## 财政与派系联动

| 行动 | 收益 | 代价 |
| --- | --- | --- |
| 内帑补军 | 快速补饷 | 宗室内廷不满 |
| 临时加派 | 国库上升 | 民心下降、地方阻力上升 |
| 商户借款 | 快速得钱 | 需要信用，违约会粮价暴涨 |
| 抄没旧党 | 钱多、爽点强 | 旧党反扑、官僚恐慌 |
| 减官俸 | 省钱 | 官僚执行下降 |
| 停赔款 | 君威上升 | 金军威压上升 |

## 玩家可见反馈

财政结果不要只显示数字。要写成可读回奏：

```text
户部回奏：本月国库入二十六万贯，内帑拨补八万贯。
禁军补发两月欠饷，士卒鼓噪稍息。
然江淮粮船因汴河水浅与盗匪阻截，实到不足六成。
开封粮价仍较上月涨三成，太学生已有聚议。
```

然后列结构化变化：

- 国库 -12 万贯。
- 内帑 -8 万贯。
- 京城粮 +9 万石。
- 禁军欠饷 -18。
- 粮价指数 +30。
- 民心 -4。

## 首发重点

MVP 只需把以下四个财政玩法做好：

1. 禁军欠饷。
2. 京城粮价。
3. 抄没追赃。
4. 江淮/京畿粮运。

这四个足以支撑守城、清算和民心压力。
