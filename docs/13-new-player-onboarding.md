# 新手引导设计

新手引导的目标不是教玩家每个按钮，而是让第一次当皇帝的人少一点：

```text
朕是谁，朕在哪，朕的钱呢？
```

玩家进入首局后，必须在 10 分钟内明白：

- 我是宋钦宗，刚梦见靖康之耻。
- 金兵快来了，汴京是核心。
- 我手里有钱、粮、民心、君威几个关键资源。
- 我需要召见大臣、查清真相、拟旨执行。
- 圣旨会变形，所以要指定承办人、钱粮来源和期限。
- 失败不是结束，而是下一条线索。

## 引导原则

1. **用剧情教系统**：不要弹一堆说明框，用第一月危机自然引导。
2. **少讲概念，多给下一步**：每条急奏给“可召见谁 / 可查什么 / 可拟什么”。
3. **先给一个小爽点**：第一回合最好能查到一条军饷线索或稳住一个城防节点。
4. **渐进揭示**：第一回合不展示所有派系、地区、诏令类型。
5. **允许犯错**：前两回合错误不要立刻城破，而是给预警和复盘。
6. **解释执行偏差**：玩家第一次旨意失败时，要清楚告诉他卡在哪一层。

## 首局引导结构

### 第 0 步：亡国梦

开局短叙事：

```text
夜半，帝梦汴京城破，宗庙蒙尘，二帝北行。
醒来时，金兵尚在河北，百官仍称“可议和缓之”。
```

引导目的：

- 建立身份。
- 建立目标。
- 建立历史改命感。

### 第 1 步：看懂顶栏

只解释 5 个资源：

- 国库：办事的钱。
- 内帑：救急的钱。
- 京城粮：围城能撑多久。
- 民心：百姓是否愿意跟你守。
- 君威：圣旨是否有人听。

提示文案：

```text
陛下此时最要紧的不是天下所有数字，而是钱、粮、人心和圣旨还管不管用。
```

### 第 2 步：第一批急奏

第一回合只给 3 条急奏：

1. 金军南下。
2. 禁军欠饷。
3. 李纲请守城。

每条急奏有明确按钮：

- 召见相关人。
- 立密令。
- 加入拟旨。

### 第 3 步：第一次召见

推荐召见李纲。

引导点：

- 大臣会给建议，但不是按钮。
- 你可以追问“要多少钱、谁来办、几日见效”。
- 他的方案会变成草案。

引导提示：

```text
问清三件事：要谁办、要多少钱、几日能见效。
```

### 第 4 步：第一次密令

禁军欠饷急奏引导玩家下密令：

```text
户部称饷已拨，禁军却称三月无钱。是否命皇城司暗查账册？
```

玩家可一键生成：

```text
密札：命皇城司三日内核验禁军军饷拨付与实发账册，勿惊动户部。
```

引导目的：

- 让玩家知道“怀疑不能直接杀，要查证”。
- 为殿前对质埋爽点。

### 第 5 步：第一次拟旨

推荐生成两道草案：

1. 任李纲临督宣化门城防。
2. 拨内帑少量补禁军急饷。

拟旨界面高亮四个必填项：

- 承办人。
- 资源来源。
- 期限。
- 风险。

提示：

```text
圣旨若没有承办人和钱粮来源，出了御案就会被官僚机器磨成空话。
```

### 第 6 步：第一次月末回奏

第一回合应设计成“有成效但不完美”：

- 李纲守城推进，宣化门略稳。
- 内帑补饷压住禁军鼓噪。
- 皇城司查到账册疑点。
- 转运财税网络开始警觉。

这能同时教会：

- 行动会有效。
- 行动有代价。
- 密令会产出下一回合爽点。

## 渐进解锁

### 第 1 回合开放

- 朝报。
- 召见。
- 密令。
- 拟旨。
- 颁诏。
- 月末回奏。

### 第 2 回合开放

- 证据。
- 殿前对质。
- 派系反应。

### 第 3 回合开放

- 朝议。
- 勤王路线。
- 粮运。

### 围城前开放

- 城门详情。
- 金宋谈判。
- 守城战报。

## 引导 UI

### 不用大段教程弹窗

使用三种轻提示：

| 类型 | 用途 |
| --- | --- |
| 御前小注 | 一句话解释当前重点 |
| 朱批高亮 | 标出下一步可点 |
| 老臣提醒 | 用角色口吻给建议 |

示例：

```text
御前小注：此事牵涉钱粮，宜先召户部或密查账册。
```

```text
老臣提醒：陛下若欲补饷，须明示钱从何来，否则户部必以“库空”为辞。
```

### 急奏卡片上的行动建议

每张急奏显示：

```text
建议：
召见：李纲 / 户部尚书 / 皇城司使
可查：禁军军饷账册
可拟：补发急饷 / 查封账册
风险：触动转运财税网络
```

### 拟旨缺口提示

当草案缺字段时：

```text
缺承办人：无人领旨，执行将大幅打折。
缺钱粮来源：户部可能以库空驳回。
缺期限：承办人会拖到下月。
缺证据：重罚可能被视为罗织。
```

## 新手保护

新手保护不是无敌，而是让第一次失败有学习空间。

### 前 2 回合保护

- 金军威压可以上升，但不直接城破。
- 主和逼宫可以预警，但不直接锁死。
- 国库可紧张，但不应因一个错误归零。
- 第一次密令失败只暴露“泄密风险”，不直接毁掉整条线。

### 首次错误解释

如果玩家下了空泛旨意：

```text
原旨：整顿禁军。
执行偏差：旨意未指定钱粮来源和主承办人，殿前司只申饬军纪，欠饷未解。
建议：补发急饷需指定金额、来源和核验人。
```

如果玩家重罚无证据：

```text
原旨：下狱某转运官。
执行偏差：证据不足，台谏质疑罗织，旧党趁机串联。
建议：可先密查账册或传相关人殿前对质。
```

## 第一次爽点设计

第二回合应能触发一个小型殿前对质：

```text
禁军欠饷疑案
证据：皇城司得东仓副册，显示拨付数与实发数不符。
对象：转运判官或户部主事。
裁断：申饬 / 下狱 / 追银 / 戴罪核账。
```

不要第一案就牵涉顶级宰执。先让玩家学会：

- 证据强弱。
- 对质流程。
- 裁断代价。
- 国库和禁军反馈。

## 失败复盘

新手局失败后，不要只显示结局。

输出：

```text
你为何没守住汴京

1. 禁军欠饷长期未解，守军战意跌至 28。
2. 京城粮价三月翻倍，民心跌至 31。
3. 罢李纲后城防无人统筹，宣化门风险升至 82。
4. 勤王诏无赏格，西军迟疑未至。

下次可尝试：
- 早查军饷账册。
- 用内帑补一次急饷。
- 任一名重臣统合城防。
- 假议和拖延时同步催勤王。
```

复盘要像老臣给你讲清局势，不像系统判你死刑。

## 可选引导模式

菜单提供：

| 模式 | 说明 |
| --- | --- |
| 初登大宝 | 默认新手引导，前三回合有提示和保护 |
| 史官简注 | 只显示轻提示，无保护 |
| 朕自有断 | 关闭引导 |

关闭引导不应关闭系统解释。执行偏差和失败原因仍然要清楚。

## 教程内容清单

首发需要写好的教程素材：

- 亡国梦开场。
- 第一回合三条急奏。
- 李纲首次召见引导。
- 皇城司密查引导。
- 拟旨缺口提示。
- 第一次月末回奏。
- 第二回合小型对质案。
- 第一次派系反扑提示。
- 第一次围城预警。
- 失败复盘模板。

## 成功标准

新手引导合格的指标：

- 80% 玩家第一回合能成功颁诏。
- 70% 玩家第二回合能理解证据和对质。
- 前三回合主动退出率低。
- 玩家失败后能说出至少一个明确原因。
- 玩家愿意开第二局尝试不同路线。

最终目标是让玩家不是被教程牵着走，而是自然产生一句话：

```text
朕懂了，这道旨不能空写，得有人、得有钱、还得防他们糊弄朕。
```
